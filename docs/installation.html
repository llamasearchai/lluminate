LlamaLluminate Installation Guide

Prerequisites

Before installing LlamaLluminate, ensure you have:

  • Python 3.8 or later
  • pip (Python package installer)

Installation via pip

The simplest way to install LlamaLluminate is using pip:

pip install lluminate

Installation from source

To install the latest development version from GitHub:

git clone https://github.com/llamasearchai/lluminate.git
cd lluminate
pip install -e .

Verifying Installation

To verify that LlamaLluminate was installed correctly, run the following in a Python interpreter:

import lluminate
print(lluminate.__version__)

Optional Dependencies

For additional features, you can install optional dependencies:

pip install lluminate[extra]

Next Steps

Now that you have LlamaLluminate installed, check out the Usage Guide to get started.
