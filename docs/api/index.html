lluminate API Reference

Complete documentation for all classes and methods in the lluminate library.

Client

The main class for interacting with the lluminate API.

Constructor

Client(api_key=None, base_url=None)

Parameters:

  • api_key (str, optional): Your API key. Defaults to None.
  • base_url (str, optional): Base URL for API requests. Defaults to None.

Example:

from lluminate import Client

client = Client(api_key="your_api_key")

process_data

process_data(data, options=None)

Parameters:

  • data (str): The data to process.
  • options (dict, optional): Additional options. Defaults to None.

Returns:

dict: The processed result.

Example:

result = client.process_data("example data")
print(result)

batch_process

batch_process(data_items, options=None)

Parameters:

  • data_items (list): A list of data items to process.
  • options (dict, optional): Additional options. Defaults to None.

Returns:

list: A list of processed results corresponding to each input item.

Example:

data_items = ["item1", "item2", "item3"]
results = client.batch_process(data_items)

for item, result in zip(data_items, results):
    print(f"Item: {item}, Result: {result}")

process_data_async

process_data_async(data, options=None, on_progress=None, on_complete=None)

Parameters:

  • data (str): The data to process.
  • options (dict, optional): Additional options. Defaults to None.
  • on_progress (callable, optional): Progress callback. Defaults to None.
  • on_complete (callable, optional): Completion callback. Defaults to None.

Returns:

Task: A task representing the async operation.

Example:

def on_progress(progress):
    print(f"Progress: {progress}%")

def on_complete(result):
    print(f"Processing complete: {result}")

task = client.process_data_async(
    "large data",
    on_progress=on_progress,
    on_complete=on_complete
)

Config

Configuration class for customizing client behavior.

Constructor

Config(timeout=60, retries=3, verbose=False)

Parameters:

  • timeout (int, optional): Request timeout in seconds. Defaults to 60.
  • retries (int, optional): Number of retry attempts for failed requests. Defaults to 3.
  • verbose (bool, optional): Enable verbose logging. Defaults to False.

Example:

from lluminate import Config

config = Config(
    timeout=30,
    retries=5,
    verbose=True
)
